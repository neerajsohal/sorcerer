Sorcerer Framework Documentation

Welcome to Sorcerer's Documentation. This guide will help you get started with Sorcerer Framework.

Introuction

Sorcerer is a free, open source wordpress theme development framework. It is highly extensible, supports SEO and widgets. It's simple enought for newbies and powerful enough for pros.

Features

  • Based on famous blueprint css framework to give you full advantage of grid based crossbrowser compatibility
  • It's a powerhouse. Sorcerer's power API maks it easy to extend it.
  • Has built in widgets and plugins for your every basic need.
  • 100% compatible with wordpress's theme development guidelines.
